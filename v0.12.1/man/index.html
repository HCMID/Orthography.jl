Orthography.validchar — Function

True if ch appears in list of all valid characters for this orthography.

validchar(ortho, ch)

ch is a string possibly including more than one Julia Char but representing a single character in the orthographic system ortho.

source
Orthography.validstring — Function

True if all chars in s are valid.

source
Orthography.tokenize — Function

Tokenize string s using the tokenizer of the given orthographic system.

The return value is a list of OrthographicTokens.

source

Tokenize citable node cn using the tokenizer of the given orthographic system.

tokenize(ortho, cn)

The return value is a list of pairings of a CitableNode and a token category. The citable node is citable at the level of the token.

source

Tokenize corpus c using the tokenizer of the given orthographic system.

tokenize(ortho, c)

The return value is a list of pairings of a CitableNode and a token category. The citable node is citable at the level of the token.

source
